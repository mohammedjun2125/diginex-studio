# **App Name**: Diginex studio

## Core Features:

- Interactive 3D Hero: Animated 3D object that reacts to cursor movements on the home page, creating an engaging visual experience.
- Animated Service Cards: Floating service cards with depth and hover animations to showcase services offered.
- Client-Side Form Submission: Contact form using EmailJS or Formspree for client-side submission without backend.
- Pricing Plan Comparison: Display pricing plans in a clear comparison table.
- Smooth Page Transitions: Implement smooth page transitions for enhanced UX.
- Cursor-Based Parallax: Incorporate cursor-based parallax effects and micro-interactions throughout the site.
- Blog Page: SEO-optimized articles categories & tags, share buttons, Clean reading experience.

## Style Guidelines:

- Primary color: White (#FFFFFF) to convey purity and cleanliness.
- Background color: Deep Purple (#301934), providing contrast and emphasizing the dark theme.
- Accent color: Lavender (#E6E6FA) for highlights and interactive elements.
- Headline font: 'Space Grotesk' (sans-serif) for headlines and short amounts of body text, 'Inter' (sans-serif) for longer text.
- Use clean, minimalist icons to represent services and features.
- Enterprise-level clean layout with a focus on readability and user experience.
- Subtle, performance-optimized animations and cursor-based interactions to enhance engagement.
